🍽️ Restaurant Menu & Ordering UI
A modern Restaurant Menu and Ordering Interface built with HTML, TailwindCSS, and Vanilla JavaScript.
This project allows users to browse food categories, add items to an order panel, adjust quantities, and see the total price dynamically.
It is designed as a front-end practice project focusing on DOM manipulation, UI layout, and interactive user experience.

✨ Features
🧾 Dynamic food menu rendering with JavaScript
🛒 Interactive Order Panel
➕ Increase or decrease item quantity
❌ Remove items from the order
💰 Automatic total price calculation
📱 Fully responsive layout
📜 Scrollable order panel for mobile
⚡ Smooth UI interactions
🎨 Modern design using TailwindCSS

🛠️ Technologies Used
HTML5
TailwindCSS
JavaScript (Vanilla JS)
DOM Manipulation


⚙️ How It Works
Food data is stored in a JavaScript array.
JavaScript dynamically renders food cards inside their category sections.
When the Order button is clicked: 
The item is added to the order panel.
Quantity can be increased or decreased.
Items can be removed.
The total price updates automatically whenever the cart changes.

📱 Responsive Design
The interface is designed to work across different devices:
Desktop
Tablet
Mobile
The order panel remains scrollable on smaller screens for better usability.
